Terminal Commands

::: Back to top :::


Find open ports

PS C:\> netstat -an | findstr "LISTENING"

PS C:\> netstat -an | findstr "ESTABLISHED"

Get IP address

PS C:\> Get-NetIPAddress -AddressFamily IPv4

Find path to program

Displays the path to git.exe:
PS C:\> (gcm git).path

Find path to Powershell

PS C:\> $pshome

Powershell Profiles

::: Back to top :::

How to set up a profile

First, test to see if a profile exists:
Test-path $profile If false, create a new one:
New-item -type file -force $profile If true, open and edit away:
Notepad $profile

Unix 'which' in Powershell

function which($name)
{
	Get-Command $name | Select-Object -ExpandProperty Definition
}
